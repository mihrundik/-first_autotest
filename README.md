# Домашнее задание №1: Первый автотест

## Цель:
Реализовать автоматический тест, используя Java + Selenium.

---

## Описание / пошаговая инструкция выполнения домашнего задания:

Для выполнения ДЗ выполните следующие шаги:

### Шаги выполнения:

1. **Переход по ссылке**:  
   Перейдите по следующей ссылке: [Форма](https://otus.home.kartushin.su/form.html)

2. **Заполнение формы**:  
   Напишите скрипт для автоматического заполнения всех полей формы:
    - Имя пользователя
    - Email
    - Пароль
    - Дата рождения
    - Уровень знания языка

3. **Проверка пароля**:  
   Добавьте логику для проверки совпадения введённого пароля и его подтверждения.

4. **Тестирование кнопки отправки**:  
   Убедитесь, что после нажатия кнопки отправки («Submit») данные корректно отображаются на странице.

---

## Критерии оценки:

| Баллы | Требования                                                                                                 |
|-------|-----------------------------------------------------------------------------------------------------------|
| +1    | Код компилируется и запускается                                                                           |
| +2    | Код запускается без дополнительных действий со стороны проверяющего                                       |
| +2    | Логин/пароль для авторизации не зашиты в код (передаются как параметры при старте)                         |
| +2    | Логи пишутся в консоль и файл                                                                             |
| +2    | Тест проходит без падений (допускается падение теста только при некорректной работе системы тестирования) |
| +1    | В репозитории нет лишних файлов (`.iml`, директория `.idea` и т.п.)                                      |

**Минимальное количество баллов:** 8
